# 🚀 AIG Admin Panel

A modern, scalable **Admin Interface Generator (AIG)** built with **Next.js**, designed to manage event-based data like **Delegates, Contacts, Abstracts, and more** with a clean UI and reusable architecture.

---

## 📌 Overview

The AIG Admin Panel is a **modular, form-driven CMS** that allows administrators to:

* Manage event-related data efficiently
* Perform full CRUD operations (Create, Read, Update, Delete)
* Maintain structured data using schema validation
* Use reusable UI components for rapid development

---

## 🛠️ Tech Stack

* **Framework:** Next.js 16 (App Router)
* **Language:** TypeScript
* **UI:** Tailwind CSS + shadcn/ui
* **Forms:** React Hook Form + Zod
* **State/Data Fetching:** SWR
* **Tables:** TanStack React Table
* **Icons:** React Icons

---

## 📂 Features

### ✅ Dynamic Form System

* Schema-based validation using **Zod**
* Reusable form components (Delegate, Contact, Abstract)
* Supports both **Create & Edit modes**

### ✅ Data Management

* Event-based routing
* Modular APIs
* Optimized data fetching with SWR

### ✅ Table UI

* Fully customizable data tables
* Status badges with visual indicators
* Inline actions (Edit, Update)

### ✅ Sheet-Based UX

* Smooth side panel forms (Add/Edit)
* Better user experience than modals



## ⚙️ Installation

```bash
git clone <your-repo-url>
cd aig-admin
npm install
npm run dev
```


## 🚀 Usage

1. Open the app in browser
2. Navigate to an event
3. Manage:

   * Delegates
   * Contacts
   * Abstracts
4. Use **Add / Edit panel** to update data

---

## 🧠 Architecture Highlights

* **Schema-driven** → Forms directly map to Zod schemas
* **Reusable Components** → One pattern for all entities
* **Separation of Concerns** → UI, validation, and API are isolated
* **Scalable Design** → Easy to add new modules
